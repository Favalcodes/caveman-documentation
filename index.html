HTML Structure

The template is based on Bootstrap Framework - http://getbootstrap.com/
The default Bootstrap grid system utilizes 12 columns, making for a 1170px wide depending on your viewport.
Below 480px viewports, the columns become fluid and stack vertically.
 
Create a .row and add the appropriate number of .col-* columns.
  
<div class="row">
        
       { Level 1 column }

        <div class="row">
            <div class="col-6"> { Level 2 } </div>
            <div class="col-6"> { Level 2 } </div>
        </div>

</div>

If you need more information, please visit this site: http://getbootstrap.com/css/#grid

CSS Structure

The organization of the CSS is one of our priorities.
These are the CSS file we're using in the template:
  
  - css/
    - style.css                - (Main CSS file) 

    - bootstrap/bootstrap.min.css - (Bootstrap Minified CSS file)
    - ionicon/ionicons.min.css     - (Ionicons Minified CSS file)

  - fonts/  - contains fonts for slick slider library

JavaScript

jQuery - is a Javascript library that greatly reduces the amount of code that you must write.
For more information, please visit http://www.jquery.com/
 
All the JavaScript libraries and files are included at the bottom of every HTML page like this: 
<!-- Main Jquery JS -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.4.1/jquery.min.js" type="text/javascript"></script>        
<!-- Popper JS -->
<script src="https://cdn.jsdelivr.net/npm/popper.js@1.16.0/dist/umd/popper.min.js" type="text/javascript"></script> 
<!-- Bootstrap JS -->
<script src="./css/bootstrap-4.5.0-dist/js/bootstrap.min.js" type="text/javascript"></script>         
<!-- Custom JS -->
<script src="./js/main.js"></script>
<!-- AOS JS -->
<script src="https://unpkg.com/aos@2.3.1/dist/aos.js"></script>

 

Fonts

To change the embedded font, please take a look in the head part of ./css/:
  
@font-face{
    font-family: sin;
    src: url(../css/Sinthya.ttf);
    font-style: normal;
    font-weight: normal;
}
@font-face{
    font-family: ibm;
    src: url(../css/IBMPlexMono-Regular.ttf);
    font-style: normal;
    font-weight: normal;
}

 

To change the font, first go to http://www.google.com/dafonts choose a font, download it and add the @font-face code to the style.css.
 
Then edit --header-font or --text-font in
  
:root {
  --header-font: 'Sinthya', serif;
  --text-font: 'IBMPlexMono-Regular', sans-serif;
  --nav-font: 'IBMPlexMono-Regular', sans-serif;
  --primary-color: #1d1d1d;
  --secondary-color: #974f10;
  --dark-color: #1f1f1f;
  --darker-color: #1d1d1d;
  --white-color: #f2f2f2;
}
